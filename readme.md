
# 책 먹는 호반우 동아리 관리 시스템

## 📖 프로젝트 개요

'책 먹는 호반우'는 독서 동아리의 전반적인 활동과 신규 회원 모집 과정을 효율적으로 관리하기 위해 구축된 웹 애플리케이션입니다. **Flask**와 **Supabase**를 기반으로 하며, 회원 관리, 위키, 면접 일정 조율, 그리고 **유전 알고리즘을 활용한 지능형 조 편성** 등 동아리 운영에 필요한 핵심 기능들을 통합 제공합니다.

-----

## ✨ 주요 기능

### 1\. 회원 및 커뮤니티 관리

  * **카카오 소셜 로그인**: 회원들은 카카오 계정을 통해 간편하게 로그인할 수 있습니다.
  * **프로필 시스템**: 각 회원은 사진, 한 줄 소개, 상세 설명 등이 포함된 개인 프로필을 작성하고 편집할 수 있습니다.
  * **내부 위키**: 동아리 내부의 정보와 지식을 공유하고 축적할 수 있는 위키 시스템을 제공합니다.
  * **마이페이지**: 회원은 자신의 출석 기록, 벌점 현황을 확인하고 다음 모임의 불참 여부를 신청할 수 있습니다.

### 2\. 신규 회원 모집 및 면접 시스템

  * **면접 이벤트 관리**: 관리자는 특정 기간에 대한 면접 이벤트를 생성하고, 면접 시간을 일괄 등록할 수 있습니다.
  * **간편한 면접 예약**: 지원자는 별도의 가입 없이 공유된 링크를 통해 접속하여 원하는 면접 시간을 직접 예약할 수 있습니다.
  * **면접관 참여**: 동아리 회원은 면접관으로서 참여 가능한 시간을 직접 선택하고 배정받을 수 있습니다.
  * **실시간 알림**: 신규 예약이 발생하면 관리자에게 텔레그램으로 실시간 알림이 전송됩니다.

### 3\. 지능형 조 편성 시스템

  * **유전 알고리즘(GA) 기반**: **성비, 새로운 만남, 발제자 분배, 회원 선호도** 등 복합적인 조건을 고려하여 최적의 조 편성을 자동으로 추천합니다.
  * **실시간 진행률 표시**: 조 편성 실행 시, 서버-센트 이벤트(SSE)를 통해 사용자에게 실시간 진행률과 예상 완료 시간을 보여줍니다.
  * **수동 기록 및 수정**: 알고리즘 결과를 수정하거나, 과거 기록을 직접 입력하고 관리하는 기능을 제공합니다.

-----

## 💻 기술 스택

  * **Backend**: Python, Flask
  * **Database**: Supabase (PostgreSQL)
  * **Frontend**: HTML, Tailwind CSS, Vanilla JavaScript
  * **Core Logic**: `deap` 라이브러리를 활용한 유전 알고리즘

-----

## 🚀 실행 방법

1.  **저장소 복제**

    ```bash
    git clone [저장소 URL]
    cd [프로젝트 폴더]
    ```

2.  **가상 환경 생성 및 패키지 설치**

    ```bash
    python -m venv venv
    source venv/bin/activate  # Windows: venv\Scripts\activate
    pip install -r requirements.txt
    ```

3.  **.env 파일 설정**
    프로젝트 루트에 `.env` 파일을 생성하고 Supabase 및 카카오 로그인 관련 환경 변수를 입력합니다.

    ```env
    FLASK_SECRET_KEY='...'
    SUPABASE_URL='...'
    SUPABASE_SERVICE_KEY='...'
    KAKAO_OAUTH_CLIENT_ID='...'
    KAKAO_REDIRECT_URI='...'
    # ... 기타 필요한 환경 변수
    ```

4.  **애플리케이션 실행**

    ```bash
    flask run
    ```

    이후 터미널에 안내된 주소(기본: `http://127.0.0.1:5000`)로 접속합니다.

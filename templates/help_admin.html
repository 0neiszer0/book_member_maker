관리자 운영 가이드

책 먹는 호반우 시스템을 효율적으로 운영하기 위한 안내서

① 학기 만들기 ② 출석 투표 ③ 발제문 수집 ④ 조 편성 ⑤ 기록 관리 ⑥ 회원 관리

① 학기 만들기 (한 학기 시작 전)

학기 단위로 세미나를 운영합니다. 시작 전에 한 번만 만들면 끝.

1
관리자 대시보드 → 세미나 출석 투표 관리 섹션
2
학기명(예: "2025-2학기"), 시작일/종료일, 한도(기본 32명) 입력 → + 새 학기 만들기
3
해당 기간 내 모든 월/목 회차가 자동 생성됨
4
링크 복사 → 카톡방에 한 번만 공유 (학기 끝까지 같은 링크 사용)
💡 학기가 늘어나면 회차 관리에서 기간 수정 시 누락된 월/목이 자동 추가됨 (기존 데이터 보존).

② 매주 출석 투표 운영

투표는 자동으로 열리고 닫힙니다. 회장이 매주 카톡방에서 묻지 않아도 됨.

  • 투표 기간: 세미나 전주 금요일 18:00 ~ 일요일 23:59 KST
  • 회원이 학번+이름으로 본인 인증 후 회차별로 참석/불참 선택
  • 투표 마감 후엔 관리자만 명단을 추가/제거 가능
1
대시보드 → 학기 줄에서 회차 관리 클릭
2
회차별로 📚 이번 회차 도서 입력 (회원에게 표시됨)
3
참석자 관리 펼쳐서 직접 추가/제거 (지각자, 카톡으로 받은 신청 등)
⚠️ 투표 마감 시점은 시스템이 자동 판단. 관리자는 마감 후에도 자유롭게 수정 가능.

③ 발제문 수집

매주 세미나 도서에 대한 발제(논의 주제)를 모읍니다.

1
대시보드 → 주간 발제문 관리 → 모임 날짜·책 제목 입력 → + 취합 링크 생성
2
생성된 링크 복사 → 카톡 공유
3
회원이 학번+이름만 입력하면 발제문 작성 가능 (PIN 불필요)
4
세미나 당일: 발제문 확인 버튼 → DOCX 다운로드 가능

④ 세미나 당일 조 편성

출석자를 자동으로 조에 분배합니다.

1
상단 메뉴 → 🧩 조 편성 진입
2
오늘 회차에 투표한 회원이 자동으로 사전 체크됨
3
현장에서 조정 (불참자 체크 해제, 신규 게스트 추가)
4
조 수, 발제자 지정 → 조 편성 시작
5
결과 확인 후 저장 → 자동으로 기록 시스템에 누적

⑤ 기록 관리

과거 활동을 학기별/전체로 조회·수정합니다.

  • 📅 학기 필터: 모든 기록 페이지 상단에서 선택. "전체"로 하면 전 기간 데이터.
  • 세미나 탭: 도서명/장르 인라인 수정. 상세 페이지에서 조 편성도 직접 편집.
  • 저장 안 한 과거 세미나: + 과거 세미나 수동 추가 버튼으로 추가
  • 벽돌책/소모임: 별도 카드로 관리, 회차 추가 시 멤버 함께 입력
  • 통계: 학기 필터 적용 시 해당 학기만의 수치 표시

⑥ 회원 관리

멤버 상태를 명확하게 구분합니다.

  • 활성: 현재 활동 중. 조 편성·통계에 포함.
  • 휴면: 일시 정지. 조 편성에서 자동 제외하지만 기록은 보존.
  • 비활성: 졸업·탈퇴 등. 기본 노출에서 빠짐.
💡 회원이 직접 회원가입하면 자동으로 활성 상태가 되며, 학번/이름이 일치해야 발제문·투표에 참여 가능.
